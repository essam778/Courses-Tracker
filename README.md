# 📚 Course Tracker - متتبع الكورسات

تطبيق ويب لتتبع تقدمك في الكورسات التعليمية مع دعم 10Themes مختلفة والوضع الداكن/الفاتح.

![Preview](https://img.shields.io/badge/HTML5-CSS3--JS-orange) ![License](https://img.shields.io/badge/License-MIT-green)

---

## ✨ المميزات المضافة

### ✅ الميزات الأساسية
- إضافة وحذف الكورسات
- تتبع التقدم عبر الـ Checkpoints
- 10Themes مختلفة (Cosmic, Amethyst, Topaz, Cyber, Aurora, Ocean, Sunset, Galaxy, Neon, Forest)
- وضع الداكن/الفاتح
- حفظ البيانات في LocalStorage (بيانات خاصة بكل مستخدم!)
- تأثيرات احتفالية عند إكمال كورس
- دعم اللغة العربية (RTL)

### 🎯 الميزات الجديدة المضافة
1. **📝 ملاحظات لكل كورس** - أضف ملاحظات وتوثيق لكل كورس
2. **🎯Goals يومية** - حددGoals تعلم يومية وتتبعها
3. **💾 تصدير البيانات** - احفظ نسخة احتياطية من بياناتك
4. **📥 استيراد البيانات** - استعد بياناتك من ملف
5. **📱 PWA** - ثبت التطبيق على هاتفك

## 💾 خصوصية البيانات

**IMPORTANT:** البيانات تُحفظ محلياً على جهازك (LocalStorage)!
- عند نشر التطبيق على GitHub Pages، كل مستخدم له بياناته الخاصة
- لا توجد قاعدة بيانات مشتركة
- استخدم زر "تصدير" لعمل نسخة احتياطية

## 🔧 طريقة التطوير

### تشغيل المشروع
فقط افتح `index.html` في المتصفح - لا حاجة لخادم!

### إضافة ميزات جديدة
1. عدل `style.css` لإضافةThemes جديدة
2. أضفFunctions جديدة في `script.js`
3. احفظ البيانات بـ `localStorage`

---

## 📱 التثبيت على الهاتف

1. افتح الرابط في متصفح Chrome على Android
2. اضغط على أيقونة القائمة (⋮)
3. اختر "إضافة إلى الشاشة الرئيسية"
4. سيظهر التطبيق كـ app مستقل!
